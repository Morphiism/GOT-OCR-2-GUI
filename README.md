## 使用方法
### 依赖
- gradio
- transformers
### 模型文件
> 如果这里面提到的文件夹你没有，那就新建一个
1. 下载到`models`文件夹中
2. 别少下载文件了
3. 选一个你喜欢的脚本打开
4. 用GUI的可以不管，不过用CLI的各位记得把要OCR的图放进`imgs`文件夹里（CLI目前只检测`.jpg`和`.png`图片）
